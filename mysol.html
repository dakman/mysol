Getting Started

✕
1. Install a wallet: Phantom (preferred) or Solflare.
2. Open this app and click Connect Wallet.
3. Pick network at top right:
Devnet for testing (free SOL faucet + devnet USDC mint),
Mainnet for real funds.
Devnet SOL: Get free SOL from faucet.solana.com.
Select Devnet, paste your wallet address, and request SOL.
Devnet USDC: Use faucet.circle.com.
Set network to Devnet, choose USDC, paste your wallet address, then mint test tokens.
4. Set daily limits + enforcement days, then click Sign & Burn Rules.
5. Deposit SOL/USDC into the vault, then withdraw within limits.
6. On Devnet, Reset Vault can end enforcement and drain funds for testing.

How It Works

✕
🔒 Immutable — rules locked on-chain forever once upgrade authority is burned.
⛔ On-chain enforcement — blockchain rejects over-limit withdrawals, not just this app.
💵 SOL + USDC — separate daily limits for each token. Both tracked independently.
🏦 Spending account — fund the vault, spend from it. Limit applies to every tx out.
🔑 No private key — vault PDA can't be imported. Only your program moves funds out.
